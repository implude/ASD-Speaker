# ASD-Speaker
ASD Project AI Speaker "Seri" Repository

# Related Repository
 - Arduino LED Controller https://github.com/implude/ASD-LED

# Installing Dependencies

1. ```pip install -r requirements.txt```
2. ```sudo apt-get install pyaudio```

# Running

1. set .env file
2. ```python3 app.py```
